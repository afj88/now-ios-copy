INVENTIA SDK - CONNECT CLIENT ASSETS
------------------------------------------------------------------------------------


If the host App embed Inventia SDK customization.
------------------------------------------------------------------------------------
Inside this folder it is possible to place your Inventia ConnectClient customization 
files like:

mobile.html
css/custom-mobile.css
... 
other custom files
...

**  File that are not a customization will downloaded from Connect Server when
    Inventia ConnectClient starts.


If the host App use Inventia SDK customization from a web server
------------------------------------------------------------------------------------
Leave this folder empty.
